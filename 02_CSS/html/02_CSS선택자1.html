태그 선택자

        - 현재 문서 내에 특정 태그를 모두 찾아 선택하는 선택자
        [작성법]
        
            태그명{
                css 코드;
            }

id(#) 태그 선택자

        HTML 문서 내에서 
        id가 일치하는 요소를 선택하는 선택자
        [작성법]
        
            #id이름 {
                css코드;
            }
