display flex - justify-content

        justify-content: div(=container)와 같은 태그를 
        수평 방향으로 정렬하고 간격을 조절하는데 사용

        display:flex 를 작성하면 justify-content:flex-start 이미 존재함
        justify-content:flex-start (기본값)

가로 정렬(수평 중앙 정렬)

display:flex justify-content:center

display:flex = 세로로 된 div 가로로 보기
justify-content:center = 가로에 있는 div 가로에서 중앙 정렬

        display:flex + justify-content:space-between
