        a 태그는 사용자가 특정 행동을 했을 경우를 3가지 경우로 나누어 설정

        a:hover = 마우스를 링크 위에 올렸을 때 스타일 설정
            주로 text-decoration : underline 밑줄추가
        a:active = 마우스를 클릭했을 때 활성화되는 스타일
